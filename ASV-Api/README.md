# API ASV 

### Descargar el api.
```sh
  git clone https://github.com/SamehArmouche/ASV.git
```
### Instalar dependencias
```sh
  cd ASV-Api
  npm i 
```
### Ejecutar la api
```sh
  npm run dev
```

### RUTAS 
#### Para obtener los municipios que que concuerdan con el prefeijo 
```url
http://localhost:3000/municipios?prefMpio=Aba
```
